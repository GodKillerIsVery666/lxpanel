# 支持

## 获取帮助

- 📖 **文档**: 查看 [Wiki](https://github.com/GodKillerIsVery666/lxpanel/wiki) 和 [docs/](docs/) 目录
- 🐛 **Bug 报告**: 提交 [GitHub Issue](https://github.com/GodKillerIsVery666/lxpanel/issues/new?template=bug_report.md)
- 💡 **功能请求**: 提交 [Feature Request](https://github.com/GodKillerIsVery666/lxpanel/issues/new?template=feature_request.md)
- 💬 **讨论**: 加入 [GitHub Discussions](https://github.com/GodKillerIsVery666/lxpanel/discussions)

## 常见问题

### 如何快速启动？
```bash
docker compose -f deploy/compose.yml up -d
```

### 如何升级？
参考 `docs/deployment.md` 的升级章节。

### 如何备份数据？
平台治理页提供状态快照和远程备份功能。
