# 路线图

## v0.1

- 登录、初始化、会话与审计。
- 系统概览、进程、服务、文件目录。
- 连接器登记和心跳协议。
- 前端基础运维工作台。

## v0.2

- Docker 状态、容器列表、镜像列表和容器动作。
- 日志根目录浏览与尾部读取。
- 多用户、RBAC、受控任务运行器和本地状态备份。
- 计划任务和自动状态备份。
- 连接器命令队列和结果回传协议。
- 可选 SQLite 状态存储与 legacy JSON 导入。
- CPU、内存、磁盘资源告警。
- 主机资产、监控趋势和 Webhook 通知渠道。
- 内置应用商店和 Docker Compose 一键部署基础。
- API Token 自动化访问。
- API Token 细粒度作用域。
- Webhook 出站白名单与通知 URL 脱敏。
- Webhook URL 加密保存与旧状态兼容读取。
- 审计筛选、导出和保留清理。
- 高风险操作审批中心，覆盖备份恢复和审计清理。
- 备份完整性校验和恢复状态保真。

## v0.3

- 远程 SSH 会话代理。
- 应用商店模板市场、版本升级和回滚。
- 多主机管理。
- 更细粒度的资源级 RBAC、审批通知和多级审批。
- 远程备份、数据库管理和防火墙/SSH 加固。
- E2E 测试、迁移脚本和安装升级体验。
