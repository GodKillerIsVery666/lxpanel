# 路线图

## v0.1

- 登录、初始化、会话与审计。
- 系统概览、进程、服务、文件目录。
- 连接器登记和心跳协议。
- 前端基础运维工作台。

## v0.2

- Docker 状态、容器列表、镜像列表和容器动作。
- 日志根目录浏览与尾部读取。
- 多用户、RBAC、受控任务运行器和本地状态备份。
- 计划任务和自动状态备份。
- 连接器命令队列和结果回传协议。
- 可选 SQLite 状态存储与 legacy JSON 导入。
- CPU、内存、磁盘资源告警。
- 主机资产、监控趋势和 Webhook 通知渠道。
- 内置应用商店和 Docker Compose 一键部署基础。
- API Token 自动化访问。

## v0.3

- 远程 SSH 会话代理。
- 应用商店模板市场、版本升级和回滚。
- 多主机管理。
- RBAC、TOTP、IP 白名单。
- 资源告警和通知渠道。
